## Project Description
**Hollow Knight: Silksong** is an informational wiki project dedicated to the sequel to the cult game *Hollow Knight*, developed by **Team Cherry**.
The project's goal is to create a structured, visually appealing, and easy-to-navigate encyclopedia about the game: characters, locations, mechanics, enemies, and other elements of the Silksong universe.

The project was implemented as part of a university course on web technologies, using modern front-end development tools.

---

## Project Goals
- Create a user-friendly and intuitive interface for viewing game information.
- Implement a responsive design for correct display on various devices.
- Use modern technologies (HTML, CSS, JavaScript, frameworks) to build dynamic pages.
- Structure content into sections (characters, enemies, world, gameplay, news, etc.).

---

## Technologies Used
- **HTML5**, **CSS3**, **JavaScript**, **Bootstrap**
- Use of **JSON** for storing and loading data
- **Git + GitHub** for teamwork and version control
